# Cookbook 


Cookbook is a recipe blog with CRUD functionality. The back-end uses Django Rest Framework.  
## Tech Stack

**Client:** HTML, JavaScript, Sass, Bootstrap

**Server:** Django, REST API


## Features

- User authentication
- CRUD functionality
- REST API
- Django models, forms, and filters
